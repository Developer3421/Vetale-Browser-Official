## What data is stored locally (on your PC)

Vetale Browser is designed to store most browser data **locally**. Examples include:
- settings and preferences
- history, bookmarks, and sessions
- tool logs (Console / DevTools)
- download history/metadata
- API keys (stored locally and encrypted)

Websites you visit can still store their own data (for example, cookies) inside the web engine, like in any browser.

---

## What may be sent to the internet

When you use online services, some information must be sent to those services to work.

Examples:
- **Web browsing:** when you open a website, your device connects to that site.
- **Search providers:** if you use an online search engine, your query is sent to that provider.
- **AI features:** if you use an AI provider that runs online, your messages are sent to that provider.
- **Voice input:** if a voice recognition feature is enabled and uses an online engine, audio may be processed by that engine.

The exact services used depend on what you enable and which providers you choose.

- clearing browsing data (history, cookies, cache) using the browser UI

- removing API keys to prevent external integrations from working

If you stop using a specific online provider, the browser will stop sending new data to it.
